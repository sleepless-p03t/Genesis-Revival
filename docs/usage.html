Usage

Downloading and installing Genesis is easy. You'll need:

  • Working internet connection
  • Kali Linux

    • Using Genesis

    Starting Genesis

    # default, animated icon display
genesis
# static icon display (no animation)
genesis -s
# no icon display
genesis -q

    Genesis is a CLI that has limited functionality by itself
    The key to Genesis's functionality resides in modules
    Using a module:

    genesis > load module

    Pressing <TAB> <TAB> will show available commands, variables, and/or available completions
